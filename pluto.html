Setup Pluto

Procedure To Setup Pluto

  • Running In Terminal Using Source Code

  • Using Executable

Steps to run pluto in Command prompt or any other Terminal.

Download and install Python 3.6 or above. Use below commands to install the required dependencies.

  • pip install pyttsx3
  • pip install SpeechRecognition
  • pip install wikipedia
  • pip install psutil
  • pip install pyjoke
  • pip install pyautogui
  • pip install pillow
  • pip install PyAudio

(Note:If you get error installing PyAudio please install using .WHL file by using pip command. Checkout videos in youtube how to install pyaudio using .whl file)

Use commands.txt file to learn to interact with Pluto. Once evrything is done, run pluto.py file using below command.

  • python pluto.py
